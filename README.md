# 🎨 CSS — Layout, Animaciones y Posicionamiento

Página web que explora las principales capacidades de CSS moderno: modelo de caja, posicionamiento relativo y absoluto, CSS Grid y animaciones. Incluye archivo de estilos externo separado del HTML.

## 🛠️ Tecnologías

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

## 📋 Contenido

- **Modelo de caja**: padding, margin, border, box-shadow
- **Posicionamiento**: relativo y absoluto
- **CSS Grid**: layout con grilla de columnas
- **Animaciones**: transiciones y movimiento con `@keyframes`
- Separación de responsabilidades HTML/CSS con archivo externo

## 🚀 Cómo ejecutar

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/MannMatias/css-layout-animations
   ```
2. Abrir `index.html` en cualquier navegador web moderno.

O visualizarlo directamente en el deployment del repositorio.
